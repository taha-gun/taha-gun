### 🩺 Pneumonia Detection from Chest X-ray Images
- Developed a medical image classification system using deep learning feature extraction
- Applied machine learning models including KNN, SVM, and Logistic Regression
- Achieved up to 85.6% test accuracy using DenseNet-based features
- Evaluated models using accuracy, precision, recall, and F1-score

### 🧠 EEG-Based Motor Imagery Classification (Graduation Thesis)
- Designed and implemented an EEG signal processing and classification pipeline
- Applied ICA, bandpass filtering (6–35 Hz), and epoch segmentation
- Used CSP for feature extraction and LDA for classification
- Implemented channel selection using functional connectivity (coherence)
- Reduced channels from 118 to ~20–30 (~70–80% reduction) while maintaining performance
